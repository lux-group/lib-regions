# lib-regions

Region information for Luxury Escapes

## Running tests

```
yarn test
```

## Camel Case

You can make everything camel case by:

```js
var libRegions = require('@luxuryescapes/lib-regions')
libRegions.setCamel()
```

## Release

```
yarn publish

```
